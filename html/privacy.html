Privacy Policy

Last update: 2024-08-11

Changelog: Initial release of privacy policy

What does Musica do?

Musica is a Discord bot designed to enhance your music experience by integrating with Last.fm. It provides features such as tracking your recent plays, viewing server-wide music statistics, and more.

What data do we store?

When you log into Musica using your Last.fm account, the following data is stored:

  • Discord Data:
    • Discord user ID
    • Presence in the server
    • Your nickname or username in that server
  • Last.fm Data:
    • Username
    • Scrobbles (plays), top artists, top albums, and top tracks
    • Registration date
    • Total scrobble count

Server Data:

When Musica joins a server, we store the following information:

  • Server name
  • Server ID
  • Members in the server who have logged into Musica
  • Nickname or username of members logged into Musica

Command Logs:

When a command is used, we log the following data:

  • Discord name or nickname of the user executing the command
  • Discord user ID
  • Server name
  • Discord server ID
  • Discord channel ID
  • The command text and options

Music Bot Scrobbling:

If you've logged into Musica, it will scrobble (store) songs played by music bots in your server to your Last.fm profile by default. You can opt out of this feature by using the /botscrobbling command.

For this feature, Musica fetches the artist and track name from messages music bots send in your server. No message content is stored or processed otherwise.

Purpose of Data Storage

We store this data to identify who is using the bot, customize command outputs, and improve the performance of our commands.

Music data and server data are used to provide server-wide leaderboards and statistics, while command logs are used for debugging and helping users with issues.

Data Retention

We retain settings data, server data, and command logs indefinitely unless you request deletion or remove Musica from your server.

Data Deletion

You can delete your user data using the /remove command. To delete server data, simply remove the bot from your server.

Users can also request data deletion by contacting us via our support server.

Infrastructure

Musica is hosted on a secure server in a reliable data center, ensuring the safety and availability of your data.

Contacting Us

If you have any security concerns or questions, you can reach out to us through our support server, direct messages, or by opening an issue on our GitHub repository.

Third-Party Services

Musica integrates with the following third-party services:

  • Last.fm

Additional Information

Musica will never share your personal data with third-party services unless explicitly stated. If you have any further questions, feel free to contact us on Discord or open an issue on GitHub.
